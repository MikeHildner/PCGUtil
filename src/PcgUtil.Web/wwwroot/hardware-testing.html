Test PCG Util with your workstation

PCG Util edits copies of your .PCG backups — the file you upload is never modified, and every edit downloads as a new file. These checks confirm that your instrument accepts an edited file and recalls the same sounds afterward. Work through them with your own gear and your own file; tick items as you go — progress is saved in this browser. Plan on about 15 minutes.

Known limitation. Sequencer songs that reference a moved program are not retargeted (set-list and combi references are). If you use songs, spot-check a song's tracks after a program reorg.
Pass criteria. The edited .PCG loads cleanly, and every patch and reference recalls the same sound as before — only the positions you intentionally changed should differ.
